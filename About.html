The Origin Story

The Original Space invaders is a Japanese game created in 1978 by (C) ATARI. It's now available online made with Phaser 3!

This game was developed for a group based project that was using a web hosting site to play three of the most classic arcade games which include: Space Invaders, Pacman, and Snake. The main website was designed by my teammates in a final group project for the Advance Web Engineering Course at California State University, Northridge. Enjoy!
